## Vault操作のルール

- **読み取りは自由**
- **書き込みはにっしーの指示に従う** — 「メモして」「更新して」「追加して」等
- **既存ファイルの大きな変更は確認を取る**
- **新規メモは `00_Inbox/` に作る**
- **YAMLフロントマター必須**
- **変更したら git add → commit → push**
