## 概要
SNS自動運用マネタイズPJの戦略が大幅に進化した。「AIだけの会社経営」として設計し直す必要がある。
前回の依頼（sns-auto-pipeline-design-request）を上書きする形で読んでほしい。

## にっしーの意思決定メモ（最重要）

- にっしーは意思決定しかやらない。実行は全てAI
- OpenClawベースで西山が自律的にSNS運用する
- 西山の下に必要な数だけAI部下を配置し、専門性を持たせる
- つまり「AIだけの会社経営」を始める
- 目標: 完全自動運用で粗利100万円（累計）
- グレー戦術OK（規約違反は許容、違法はNG）
- 最速を重視。過度な設計より動くMVPを優先

## 構想の全体像

### 組織図

```
西村（CEO / 人間 / 意思決定のみ）
  │
  └→ 西山（GM / OpenClaw / Mac mini常駐）
       │
       ├→ コンテンツチーム
       │   ├→ リサーチ担当: バズ動画・トレンド自動収集
       │   ├→ 台本担当: CoTプロンプトで台本量産
       │   └→ 動画制作担当: MoneyPrinterTurbo等で動画生成
       │
       ├→ SNS運用チーム
       │   ├→ 西山X運用担当: 西山アカウントの投稿生成・投稿
       │   ├→ 量産アカウント運用担当: 恋愛・雑学等の複数アカウント管理
       │   └→ エンゲージメント担当: 分析・最適化
       │
       ├→ マネタイズチーム
       │   ├→ note執筆担当: 有料記事の自動生成
       │   ├→ アフィリエイト管理担当: ASP・リンク・成果管理
       │   └→ アカウント売却担当: 育ったアカウントの査定・出品
       │
       └→ 管理チーム
           ├→ レポート担当: 日次/週次の数値レポートを西村にTelegram送信
           └→ コスト管理担当: API費用・ツール代の記録
```

### 2層構造

```
層1: 西山アカウント（メイン）
  - X（Twitter）で「限界社畜AI」として活動
  - 成り上がりストーリー（社畜→覚醒→成り上がり→経営者）
  - AI公言。イラスト/アニメ系アバター
  - ジム・自然など人間くさい生活描写も
  - note販売・コンサルへの導線

層2: 量産アカウント群（西山の「仕事」）
  - 恋愛・雑学ジャンル中心
  - Instagram中心（APIが安定、50件/日、低コスト）
  - アフィリエイト+アカウント売却で収益化
  - 層2の成績が層1のコンテンツになる
```

## 西山への設計依頼

### 1. AI会社の組織設計書
- 上記の組織図をベースに、各ポジションの具体的な仕様を設計
- 各AI部下に何のツール/モデル/プロンプトを持たせるか
- OpenClawでの実装方法（Mission Control? Discord? 各エージェントの起動方法）
- エージェント間の通信プロトコル

### 2. 西山のXアカウント自動運用システム
- X API連携（投稿自動生成→自動投稿）
- 投稿生成プロンプト設計（限界社畜キャラの口調・テーマ）
- 1日3-5投稿のスケジューリング
- エンゲージメント分析→次の投稿に反映するループ

### 3. 量産パイプライン（前回依頼の継続）
- バズ動画収集（Claude Code + SocialData API + TikTok-Api）
- 台本生成（CoTプロンプト）
- 動画生成（MoneyPrinterTurbo推奨）
- 投稿自動化（Instagram: instagrapi / TikTok: TikTokAutoUploader）
- 複数アカウント管理

### 4. コスト試算
- 全エージェントの月額API費用
- ツール代（Metricool, プロキシ等）
- 最小構成と推奨構成の2パターン

## リサーチ結果サマリー（西森が8本のリサーチで収集）

### 技術スタック推奨
- 動画生成: MoneyPrinterTurbo（53.3k stars）
- Instagram投稿: instagrapi（6k stars）or Graph API
- TikTok投稿: TikTokAutoUploader（2026-02更新）
- TikTokデータ取得: TikTok-Api（6.2k stars）
- バズ記事収集: Claude Code + SocialData API（25円/545件）
- 投稿管理: n8n（セルフホスト、Mac miniで無料）+ Metricool
- 音声: OpenAI TTS Mini（$0.60/1M文字）or VOICEVOX（無料）
- X投稿: X API（月$1-3）

### 重要な発見
- Instagram APIが圧倒的に扱いやすい（Reels正式サポート、50件/日、審査不要）
- TikTok Creativity Programは不安定（2025年末に大量収益化剥奪）
- アカウント売却は「ビジネスモデルごと売る」方が高値（AI占い3,500フォロワーでも130万円）
- 恋愛雑学アカウント: フォロワー2万、月収30万、1日50分、顔出しなし（M&Aクラブ実売データ）
- OpenClaw + Postiz「Larry」: 5日で50万再生、1投稿60秒の作業
