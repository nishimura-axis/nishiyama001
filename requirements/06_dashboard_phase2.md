# ダッシュボード Phase 2 設計書

## 概要
影分身ダッシュボードをスマホファースト＋タスク管理＋コスト表示に進化させる。

## ターゲットユーザー
にっしー（経営システム本部長）。毎朝スマホで10秒で全体把握。外出先からも確認。

---

## 画面構成

### 1. ホーム画面（リニューアル）

**上段: にっしーのタスク**
- 4号（nishiyama004）のtasks.jsonから取得
- カード型、縦スクロール
- タップでチェック（完了トグル）→ API経由でtasks.json更新
- 期限表示: 過ぎたら🔴、今日なら🟡、余裕あり🟢
- タスク追加ボタン（+）→ インラインフォームで即追加

**中段: 影分身ステータス**
- 各エージェントのカード（コンパクト）
  - 名前 + 絵文字
  - ステータス: 🟢稼働中 / 🟡待機中 / 🔴ブロッカーあり
  - 現在のタスク（1行）
  - 今日のコスト（円表示）
- タップで詳細ページへ

**下段: 要対応**
- にっしーのアクション必要なもの
  - 未回答の質問
  - レビュー待ち
  - ブロッカー報告

### 2. タスク管理ページ（新規: /tasks）

**にっしー本人のToDoリスト**
- 一覧表示（未完了が上、完了が下）
- 各タスク:
  - チェックボックス（タップで完了）
  - タスク名
  - 期限（あれば）
  - 編集ボタン（タップでインライン編集）
- フィルタ: 全て / 未完了 / 完了
- ソート: 期限順 / 作成順
- 新規追加フォーム（常時表示、上部）

**API設計:**
- `GET /api/tasks` — 4号のtasks.json読み取り
- `POST /api/tasks` — タスク追加
- `PATCH /api/tasks/[id]` — 更新（完了・編集）
- `DELETE /api/tasks/[id]` — 削除

### 3. コストページ（新規: /costs）

**各影分身の利用コスト**
- 期間切替: 当日 / 前日 / 今週 / 前週 / 当月 / 前月 / 月別推移
- エージェント別:
  - トークン使用量（input / output）
  - 円換算コスト
  - セッション数
- 合計コスト表示（上部に大きく）
- 月別推移グラフ
- **ホーム画面のエージェントカードには当月合計コストをデフォルト表示**

**コスト計算:**
- Opus: Input $15/1M tokens, Output $75/1M tokens
- 為替: 150円/$（or 環境変数で設定可能）

**データソース:**
- OpenClaw Gateway API（セッションログ）
- cronのrun履歴（`openclaw cron runs`）

### 4. エージェント詳細ページ（改善）

**追加項目:**
- 今日のコスト表示
- セッション使用率（トークン / コンテキスト上限）
- 最終アクティブ時刻

## スマホファースト設計

### レイアウト
- max-width: 100vw（PCでも中央寄せ max-w-lg）
- カードは横幅いっぱい、角丸、影あり
- タップ領域は最低44x44px
- フォントサイズ: 本文16px、ラベル14px、数値24px

### ナビゲーション
- 下部タブバー（固定）:
  - 🏠 ホーム
  - ✅ タスク
  - 💰 コスト
  - 📋 カンバン
- ヘッダーはコンパクト（ロゴ + ページ名のみ）

### インタラクション
- スワイプでタスク完了 or 削除（将来）
- プルダウンでリフレッシュ（将来）
- チェックボックスタップで即時反映（楽観的更新）
