🧠 太一の第二の脳セットアップ

Obsidian × Claude Code で「AIが管理する情報倉庫」を作る

1 Claude Max に契約する ⏱️ 5分

  1. ブラウザで claude.ai を開く
  2. 「Sign Up」からアカウントを作成（会社メールでOK）
  3. ログイン後、左下のアイコン →「Upgrade」
  4. Max（月$100 ≒ 約15,000円）を選択
  5. クレジットカード情報を入力して契約完了
⚠️ ProではなくMaxを選んでください。Proだと使用量の上限にすぐ到達します。
💳 支払い方法がわからない場合は西村に聞いてください。

2 Obsidianをインストール ⏱️ 5分

  1. ブラウザで obsidian.md を開く
  2. 「Get Obsidian for free」をクリック
  3. Mac用をダウンロード・インストール

3 Vaultを作成する ⏱️ 3分

  1. Obsidianを起動
  2. 「Create new vault」をクリック
  3. Vault name: MyVault（好きな名前でOK）
  4. Location: 「書類」（Documents）フォルダを選択
  5. 「Create」をクリック
💡 作成したらObsidianは一旦閉じてOK。この後はClaude Codeで操作します。

4 GitHubアカウントを作成する ⏱️ 5分

  1. ブラウザで github.com を開く
  2. 「Sign up」をクリック
  3. メールアドレス（会社メールでOK）、パスワード、ユーザー名を入力
  4. メールに届いた認証コードを入力

5 Claude Desktopアプリをインストール ⏱️ 10分

① インストール

  1. ブラウザで claude.ai/download を開く
  2. Mac用のアプリをダウンロード・インストール

② 起動・ログイン

  1. アプリを起動
  2. STEP 1で作ったアカウントでログイン
  3. ブラウザが開いたら「Authorize」をクリック

③ Codeタブを開いてVaultを選択

  1. 上部メニューから 「Code」タブ をクリック
  2. 「Select folder」で STEP 3 で作った Vaultフォルダ を選ぶ
    （書類 → MyVault）
  3. 「このフォルダを信頼しますか？」→ 「はい」
💡 「Code」タブがClaude Codeです。ここから日本語で指示を出すだけで、ファイル作成・編集・Git操作すべてをAIが実行します。
⚠️ 「Code」タブが見つからない場合 → アプリ上部メニュー「Claude」→「Check for Updates」で最新版に。

6 Claude Codeで初期セットアップ 🎉 ここからAIにお任せ！

Claude Codeの入力欄に、以下をコピペして送信してください。

# 第二の脳 初期セットアップ

私のObsidian Vaultを「第二の脳」として構築してください。

## 1. フォルダ構造の作成

以下のフォルダを作成してください：

```
00_Inbox/          ← とりあえず放り込む場所
07_DailyLog/       ← 日々のログ（自動蓄積）
10_Goals/          ← 目標管理
20_Context/        ← 業務の前提情報（AIの知識ベース）
30_Projects/       ← プロジェクト管理
40_Knowledge/      ← 普遍的なノウハウ
50_Literature/     ← 読書・記事メモ
60_Templates/      ← テンプレート
```

各フォルダに .gitkeep を作成してください。

## 2. CLAUDE.md の作成

~/.claude/CLAUDE.md に以下の内容を保存してください（ディレクトリがなければ作成）：

```markdown
# Global Rules

## 言語
- 日本語で応答する

## 基本方針
- 推測・創作を禁止。不明点は必ず質問する
- 複数の解釈が可能な場合は確認する
- セッション開始時にメモリファイルを読んでから応答
- ユーザーの訂正や好みは即座に保存

## ワークフロー: Plan → Execute
1. まず計画を立てて確認する
2. 承認されてから実行する
3. スコープが逸れたら計画に戻る

## 作業の可視化
- 各ステップの前後でフェーズを表示する
- 失敗したら正確なエラーを表示する

## コーディング原則
- 既存のアーキテクチャとパターンに従う
- 依頼されていない機能追加やリファクタリングはしない
- 秘密情報をコードにハードコードしない
```

## 3. Git初期設定とGitHub連携

私にGitHubのユーザー名とメールアドレスを聞いてから、以下を実行してください：
- `gh` コマンドがなければ `brew install gh` でインストール
- `gh auth login` でブラウザ認証
- リポジトリ名: MyVault（private）で作成
- 全ファイルをcommitしてpush（メッセージ：「初期セットアップ完了」）

## 4. 動作確認

以下を確認して結果を教えてください：
- [ ] フォルダ構造が正しく作成されている
- [ ] CLAUDE.mdが~/.claude/に保存されている
- [ ] GitHubにプライベートリポジトリが作成されている
- [ ] pushが成功している

よろしくお願いします！
⚠️ AIが途中で「実行してよいですか？」と聞いてきます。すべて「はい」でOK。
💡 GitHub認証時にブラウザが開きます。「Authorize GitHub CLI」の画面で緑のボタンをクリック。
