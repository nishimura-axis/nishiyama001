🧠 太一の第二の脳セットアップ

Obsidian × Claude Code × GitHub で「AIが管理する情報倉庫」を作る

1 Claude Max に契約する ⏱️ 5分

  1. ブラウザで claude.ai を開く
  2. 「Sign Up」からアカウントを作成（会社メールでOK）
  3. ログイン後、左下のアイコン →「Upgrade」
  4. Max（月$100 ≒ 約15,000円）を選択
  5. クレジットカード情報を入力して契約完了
⚠️ ProではなくMaxを選んでください。Proだと使用量の上限にすぐ到達します。
💳 支払い方法がわからない場合は西村に聞いてください。

2 Obsidianをインストール ⏱️ 5分

  1. ブラウザで obsidian.md を開く
  2. 「Get Obsidian for free」をクリック
  3. Mac用をダウンロード・インストール

3 Vaultを作成する ⏱️ 3分

  1. Obsidianを起動
  2. 「Create new vault」をクリック
  3. Vault name: MyVault（好きな名前でOK）
  4. Location: 「書類」（Documents）フォルダを選択
  5. 「Create」をクリック
💡 作成したらObsidianは一旦閉じてOK。この後はClaude Codeで操作します。

4 GitHubアカウントを作成する ⏱️ 5分

  1. ブラウザで github.com を開く
  2. 「Sign up」をクリック
  3. メールアドレス（会社メールでOK）、パスワード、ユーザー名を入力
  4. メールに届いた認証コードを入力
💡 ユーザー名はメモしておいてください。STEP 7で使います。

5 Claude Desktopアプリをインストール ⏱️ 10分

① インストール

  1. claude.ai/download からMac用アプリをダウンロード
  2. ダウンロードしたファイルを開いてインストール

② 起動・ログイン

  1. アプリを起動
  2. STEP 1で作ったアカウントでログイン
  3. ブラウザが開いたら「Authorize」をクリック

③ Codeタブを開いてVaultを選択

  1. 上部メニューから 「Code」タブ をクリック
  2. 「Select folder」で STEP 3 で作った Vaultフォルダ を選ぶ
    （書類 → MyVault）
  3. 「このフォルダを信頼しますか？」→ 「はい」
💡 「Code」タブがClaude Codeです。日本語で指示を出すだけで、ファイル作成・Git操作すべてAIが実行します。
⚠️ 「Code」タブが見つからない → アプリ上部メニュー「Claude」→「Check for Updates」で最新版に。

7 GitHub連携 + 初回push ⏱️ 10分

VaultをGitHubにバックアップします。以下をClaude Codeに送信：

以下の作業をお願いします。

## 1. GitHub CLI（gh）のインストールと認証

- `gh` コマンドがなければ `brew install gh` でインストール
- `gh auth login` でブラウザ認証（「GitHub.com」「HTTPS」「Login with a web browser」を選択）

## 2. VaultをGitで管理開始

- `git init` でこのフォルダをGit管理下にする
- `.gitignore` を作成（以下の内容）：
```
.obsidian/workspace.json
.obsidian/workspace-mobile.json
.obsidian/plugins/obsidian-git/data.json
.trash/
.DS_Store
```
- git config でユーザー名とメールを設定（私に聞いてから設定して）

## 3. GitHubにプライベートリポジトリを作成

- `gh repo create MyVault --private --source=. --remote=origin`
- 全ファイルを add → commit（メッセージ:「初期セットアップ完了」）→ push

## 4. 動作確認

- [ ] `git remote -v` で origin が設定されている
- [ ] `git log` でコミットが1件ある
- [ ] GitHubの自分のリポジトリページにファイルが表示される

確認結果を教えてください。
💡 GitHub認証時にブラウザが開きます。「Authorize GitHub CLI」で緑のボタンをクリック。

8 Obsidian Git プラグインで自動同期 ⏱️ 5分

Obsidianを開くだけで自動的にGitHubと同期されるようにします。

① プラグインをインストール

  1. Obsidianを開く
  2. 左下の ⚙️（設定）をクリック
  3. 「コミュニティプラグイン」→「コミュニティプラグインを有効にする」をON
  4. 「閲覧」をクリック
  5. 検索欄に 「Obsidian Git」 と入力
  6. 「Obsidian Git」をインストール → 「有効にする」

② 自動同期の設定

  1. 設定 → コミュニティプラグイン → Obsidian Git の ⚙️
  2. 以下を設定：
    • 📌 Vault backup interval (minutes): 10（10分ごとに自動push）
    • 📌 Auto pull interval (minutes): 10（10分ごとに自動pull）
    • 📌 Pull updates on startup: ON（起動時に最新を取得）
    • 📌 Push on backup: ON（バックアップ時に自動push）
💡 これで10分おきにGitHubと自動同期されます。複数のPC間でVaultが常に最新状態になります。
