🧠 はなちゃんの Claude Code セットアップ

既存のObsidian VaultにClaude Codeからアクセスできるようにする

🎯 ゴール

今使っているObsidian Vault を、Claude Code（AI）から読み書きできるようにする。
Obsidian・GitHub は導入済みなので、Claude CodeのセットアップだけでOK。

1 Claude Max に契約する ⏱️ 5分

既に契約済みならスキップ。

  1. ブラウザで claude.ai を開く
  2. アカウント作成 or ログイン
  3. 左下のアイコン →「Upgrade」→ Max（月$100）を選択
⚠️ ProではなくMaxを選んでください。Proだと使用量の上限にすぐ到達します。
💳 支払いがわからなければ西村に聞いてください。

2 Claude Codeを起動してVaultを開く ⏱️ 10分

Cursorは導入済みなので、2つの方法から選んでください。

方法A: Claude Desktop アプリ（おすすめ）

  1. claude.ai/download からアプリをダウンロード・インストール
  2. アプリを起動してClaudeアカウントでログイン
  3. 上部メニューの 「Code」タブ をクリック
  4. 「Select folder」で Obsidian Vaultのフォルダ を選ぶ
  5. 「このフォルダを信頼しますか？」→「はい」
💡 「Code」タブが見つからない → アプリ上部「Claude」→「Check for Updates」で最新版に。

方法B: Cursorのターミナルから起動

  1. Cursorのターミナル（Ctrl + `）で以下を実行：
npm install -g @anthropic-ai/claude-code

インストール後、Vaultフォルダに移動して起動：

cd ~/Documents/MyVault && claude

※ Vaultの場所が違う場合はパスを変えてください

⚠️ npmが見つからない → nodejs.org からNode.jsを先にインストール。

3 CLAUDE.md を設定する 🎉 AIにお任せ！

Claude Codeが起動したら、以下をコピペして送信してください。
AIの動き方を決める設定ファイルが自動で作られます。

以下の作業をお願いします。

## 1. CLAUDE.md（グローバル設定）を作成

~/.claude/CLAUDE.md に以下を保存してください（ディレクトリがなければ作成）：

```markdown
# Global Rules

## 言語
- 日本語で応答する

## 基本方針
- 推測・創作を禁止。不明点は必ず質問する
- 複数の解釈が可能な場合は確認する
- セッション開始時にメモリファイルを読んでから応答
- ユーザーの訂正や好みは即座に保存

## ワークフロー: Plan → Execute
1. まず計画を立てて確認する
2. 承認されてから実行する
3. スコープが逸れたら計画に戻る

## 作業の可視化
- 各ステップの前後でフェーズを表示する
- 失敗したら正確なエラーを表示する

## コーディング原則
- 既存のアーキテクチャとパターンに従う
- 依頼されていない機能追加やリファクタリングはしない
- 秘密情報をコードにハードコードしない
```

## 2. 動作確認

以下を確認して結果を教えてください：
- [ ] CLAUDE.md が ~/.claude/ に保存されている
- [ ] 現在のフォルダがObsidian Vaultである（.obsidian/ フォルダがある）
- [ ] Vault内のファイル一覧を表示

よろしくお願いします！
⚠️ AIが「実行してよいですか？」と聞いてきたら「はい」でOK。

🎉 セットアップ完了！
